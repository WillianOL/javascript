Aprendendo sobre o DOM

O que é o DOM

O DOM é uma acrônimo para Documents Object Model(Modelo de objetos para documentos). Ele é um conjunto de objetos que da acesso aos componentes de um web-site.

Seleções de DOM

As seleções do dom podem ser de varias maneiras, eles servem basicamente para selecionar um objeto dentro do codigo HTML ou chamar uma variavel.

  • Por Marca
  • Por Id
  • Por Nome
  • Por Classe
  • Por Seletor
